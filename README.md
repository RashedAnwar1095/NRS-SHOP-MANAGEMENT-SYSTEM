# NRS-SHOP-MANAGEMENT-SYSTEM
NRS Shop Management System is a console-based mini CRM application written in C.
It helps small shop owners manage products, customers, and sales records using file handling.
This project demonstrates how real-world business operations can be implemented using C programming concepts such as structures, functions, and file I/O.
